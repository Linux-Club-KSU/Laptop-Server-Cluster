# Documentation README

## Tools
- Typst CLI (0. 14.2 or higher)
- An SVG image editor
- A means to view PDFs

## Usage
One of the following methods should be tested, and both should work.

### Typist CLI Direct
- Insert the following command **Inside the directory**
    ```bash
    typst compile main.typ
    ```

- view the pdf returned

### VSCode Extension
- Find and install the "Tinymist Typist" extension

- Press the pdf button at the top of the "main.typ" file
